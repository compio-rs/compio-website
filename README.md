# Compio Website

## Development

```shell
# Install dependencies
pnpm install

# Start the development server
pnpm dev
```

## Build

```shell
# build for production environment
pnpm build
```

## Advanced

```shell
# preview the release environment effect
pnpm preview

# code check
pnpm lint

# code auto fix
pnpm format
```
